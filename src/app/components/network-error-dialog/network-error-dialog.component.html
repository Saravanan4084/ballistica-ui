Connection Error

Failed to connect to the server. Possible reasons:

  • The selected server is offline.
  • The proxy is not responding.

You can:

  • Wait and retry after a few minutes.
  • Ensure the server is online by visiting http://serverIP:PORT.
  • Navigate to Switch server and select another server.
  • Check the console for more details.
  • Change the proxy if necessary (add your own if you feel so).

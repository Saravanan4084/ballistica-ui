Welcome to BCS Server Manager

Manage/Visualize Your BombSquad Server Efficiently

Your one-stop solution for managing your BombSquad server. Here you can:

  • View live status of the server, including current players, scores, and the map running on the server.
  • Check the leaderboard and subscribe to notifications for specific players to get web push notifications when they come online.
  • Access server management tools to configure server settings, manage roles and permissions, and view all server logs in one place.

Let's get started by adding your server details

  • Navigate to switch server page.
  • Click add host and enter game server IP PORT.
  • Thats it, this IP PORT will be saved in your browser, to let you navigate across servers.
  • If you are admin of server, navigate to Admin Tools and login with password.
